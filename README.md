### How to use:

#### 1. Install the package

```bash
go get github.com/JuanBrotenelle/MyLogger@v0.1.1
```

#### 2. Create a logger

```go
log := NewLogger("INFO")
```

> [!IMPORTANT]
> The following levels can also be used:
> "INFO", "ERROR", "WARN", "DEBUG"

#### 3. Use

```go

l := NewLogger("DEBUG")

l.Debug("Debug message")

l.Info("Info message")

l.Warn("Warn message")

l.Error("Error message")

l.Fatal("Fatal message")
```
